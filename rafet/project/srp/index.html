SRP Smart Recource Planning

Project Description

Developing a web-based MRP for a furniture manufacturer that optimizes inventory utilization. App has a user friendly interface for customer ordering and for factory floor to organize shipments and view available inventory.

Project Details & Features

  • Web Application
  • One Page, Role Based MRP/ERP APP
  • AngularJs + MVC
  • HTML5 + PHP5 + mysql DB + JSON
  • Agile + UML2
  • XAMPP Stack
  • User Friendly + Responsive
  • Hight Performance in IOS + Android
  • RESTful web services, Data Validation, Authentication
  • Application was recognized as the best project in the Information Technology department at the university of ISET Sidi Bouzid.
  • Language : English
